# Reflection: How AI Assisted Me

During backend development, AI helped me define and refine **Fluent API validations** for my entities so the model rules were consistent and explicit.

It implemented the db seeder method upon my request generating the products, and later extended seeding when categories and product relationships were added.

AI also supported setup tasks such as **configuring CORS** and **setting up the DbContext** correctly for the API.

In terms of debugging, initially the HTTP test method was failing. AI helped me isolate the issue and identify that the **appsettings file was misplaced in the WebAssembly project**, which was causing incorrect configuration resolution.

AI also helped diagnose and fix a bug related to the price localization. It assisted in identifying formatting/culture mismatches and applying a consistent approach so price display matched expected locale behavior.

On the frontend side, AI assisted me in improving presentation by polishing the UI using Bootstrap classes.
